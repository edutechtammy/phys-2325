# Supporting Interaction Design Ideas

This document provides detailed descriptions of interactive activities proposed to enhance student learning for PHYS-2325 Module 1, Lesson 1. These activities are intended for stakeholder review and decision-making.

---

## 1. Error Propagation Calculator

**Description:**  
A web-based tool where students input measured values and their uncertainties (e.g., length = 12.3 ± 0.1 cm, width = 4.56 ± 0.05 cm). The calculator guides students to select the type of calculation (addition, subtraction, multiplication, division, or custom formula). It then computes the result and the propagated uncertainty, showing step-by-step reasoning.

**Student Interaction:**  
- Enter measured values and uncertainties in provided fields.  
- Choose the operation or enter a formula.  
- Click “Calculate” to see the result and a breakdown of the uncertainty calculation.  
- Optionally, try “challenge” problems or receive feedback on common mistakes.

**Learning Impact:**  
Students see how uncertainties combine, reinforcing the importance of error analysis in scientific measurement.

---

## 2. Dimensional Analysis Sandbox

**Description:**  
An interactive “sandbox” where students build equations by dragging and dropping variables (e.g., mass, velocity, acceleration) and mathematical operations. The tool checks the dimensional consistency of the equation in real time, highlighting errors and providing hints.

**Student Interaction:**  
- Drag variables and operations onto a workspace to construct equations.  
- Instantly see if the equation is dimensionally correct (e.g., green check for correct, red X for incorrect).  
- Receive hints or explanations for errors.  
- Explore “challenge” tasks, such as fixing incorrect formulas or deriving units for a given equation.

**Learning Impact:**  
Students develop intuition for unit consistency and learn to spot errors before they become problems in calculations.

---

## 3. Measurement Tool Simulator

**Description:**  
A virtual lab where students use simulated tools (ruler, caliper, micrometer) to “measure” objects of various sizes. The interface mimics real measurement tools, including reading scales and estimating uncertainty.

**Student Interaction:**  
- Select a tool and an object to measure.  
- Use mouse or touch to align the tool and take a measurement.  
- Record the measured value and estimate the uncertainty.  
- Compare results across tools to see differences in precision.  
- Receive feedback on tool choice and measurement accuracy.

**Learning Impact:**  
Students gain hands-on experience with measurement, understand the importance of tool selection, and see how uncertainty arises in practice.

---

These descriptions are intended to help stakeholders visualize each activity’s purpose, student experience, and educational value. Further expansion or visual mockups can be provided upon request.
