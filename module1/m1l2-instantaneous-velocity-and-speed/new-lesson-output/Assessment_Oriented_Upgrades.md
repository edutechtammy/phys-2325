## 1. Formative Assessment Upgrades

### A. Conceptual Check-In Questions (Embedded in Lesson)

#### Question Set 1: Limit Process Understanding
**Question:** Which statement best describes why we need the limit process in defining instantaneous velocity?

a) It makes calculations easier than using average velocity
b) It allows us to find velocity at an exact instant when Δt becomes infinitesimally small
c) It eliminates the need for calculus in physics
d) It only applies to objects moving in straight lines

**Correct Answer:** b
**Feedback:** The limit process is fundamental because it captures the essence of "instantaneous" - velocity at a precise moment rather than over an interval.

#### Question Set 2: Measurement Precision
**Question:** A GPS system reports position with ±3m accuracy every 1 second. What is the primary limitation for calculating instantaneous velocity?

a) The time interval is too large
b) The position uncertainty is too high
c) Both time and position limitations affect accuracy
d) GPS cannot measure velocity directly

**Correct Answer:** c
**Feedback:** Both temporal resolution and spatial precision affect velocity calculations. Professional applications require consideration of both factors.

### B. Interactive Problem Solving (Canvas Compatible)

#### Drag-and-Drop Velocity Calculation
Students arrange the steps for calculating instantaneous velocity from position function x(t) = 4t³ - 2t² + 5:

**Correct Sequence:**
1. Write the general limit definition: v(t) = lim[Δt→0] [x(t+Δt) - x(t)]/Δt
2. Substitute the position function
3. Expand x(t+Δt) using algebra
4. Simplify the difference quotient
5. Take the limit as Δt approaches zero
6. Result: v(t) = 12t² - 4t

#### Multiple Choice with Mathematical Reasoning
**Problem:** A particle's position is given by x(t) = sin(2πt). At t = 0.25s, the instantaneous velocity is:

a) 0 m/s
b) 2π m/s  
c) -2π m/s
d) π m/s

**Solution Process Required:** Students must show derivative calculation and substitution.

## 2. Summative Assessment Upgrades

### A. Multi-Part Engineering Applications

#### Assessment 1: Automotive Crash Analysis
**Context:** Professional crash reconstruction scenario

**Part A (15 points):** Calculate instantaneous velocity at impact
- Given: Position data from accelerometer: x(t) = 25t - 4.9t²
- Find: Velocity at t = 2.5s (moment of impact)
- Required: Complete calculus solution with units

**Part B (10 points):** Uncertainty analysis
- Measurement uncertainty: ±0.1s in time, ±0.5m in position
- Calculate: Uncertainty propagation in velocity calculation
- Required: Error analysis methodology

**Part C (10 points):** Reference frame considerations
- Determine velocity relative to: (1) ground, (2) other vehicle
- Given: Other vehicle velocity = 15 m/s
- Required: Vector addition and reference frame transformation

**Rubric:**
- Mathematical accuracy (20 points)
- Physical reasoning (10 points)
- Professional communication (5 points)

#### Assessment 2: GPS Navigation Optimization
**Context:** Developing navigation algorithms for autonomous vehicles

**Part A (20 points):** Velocity calculation from discrete data
- Given: GPS coordinates at 0.1s intervals with realistic noise
- Task: Implement three different velocity calculation methods
- Compare: Simple difference, moving average, polynomial fitting

**Part B (15 points):** Algorithm performance analysis
- Evaluate: Accuracy vs. computational efficiency trade-offs
- Consider: Real-time processing constraints
- Required: Professional engineering judgment

### B. Conceptual Synthesis Problems

#### Assessment 3: Historical Physics Analysis
**Problem (25 points):** Compare Newton's and Leibniz's approaches to instantaneous motion

**Requirements:**
- Mathematical comparison of their methods
- Historical context and scientific impact
- Modern interpretation using limit concepts
- Critical evaluation of each approach's strengths

**Assessment Criteria:**
- Historical accuracy (8 points)
- Mathematical understanding (10 points)
- Critical thinking (7 points)

#### Assessment 4: Measurement Philosophy
**Problem (20 points):** "True instantaneous velocity cannot be measured, only calculated." Evaluate this statement.

**Required Elements:**
- Discussion of measurement vs. calculation distinction
- Heisenberg uncertainty principle considerations
- Practical implications for engineering applications
- Personal position with supporting arguments

**Structure:** Teams of 3-4 students tackle complex multi-phase problems
**Assessment:** Individual accountability + team problem solution
**Example:** Design a velocity measurement system for a specific application

**Individual Component (15 points):**

**Team Component (25 points):**
- Solution accuracy and completeness
- Creative problem-solving approach
- Professional presentation quality

Students develop 3-4 case studies throughout the lesson demonstrating:

4. Communication to different audiences (technical/non-technical)

**Portfolio Grading:**
- Mathematical accuracy (40%)
- Application relevance (30%)
- Communication quality (20%)
- Critical thinking depth (10%)

### C. Technology-Enhanced Assessment

#### Virtual Laboratory Reports
Using lesson simulations and interactive tools:
- Data collection from virtual experiments
- Analysis using proper mathematical techniques

**Lab Report Components:**

2. Data collection and analysis (25 points)
3. Results interpretation (15 points)
