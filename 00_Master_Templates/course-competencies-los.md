## Module–Competency Mapping

Each **Course Competency (CC)** maps directly to one course module.  
The CC number prefix matches the module number (e.g., CC1.x = Module 1, CC2.x = Module 2).  
A ★ in lesson HTML indicates that a specific page addresses that LO.
